# Installation
1.clone Repository & Install Packages

pip install -r requirements.txt

2.Setup Virtualenv

virtualenv env
source env/bin/activate

3.Migrate & Start Server

python manage.py makemigrations
python manage.py migrate
python manage.py runserver
